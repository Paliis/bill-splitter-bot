# Юридичні документи (чернетки)

Файли **[PRIVACY.md](./PRIVACY.md)** та **[TERMS.md](./TERMS.md)** — **не** є юридичною консультацією. Це структуровані чернетки для сервісу на кшталт Bill Splitter (облік спільних витрат, без прийому чи зберігання грошей користувачів).

## Перед публікацією обов’язково

1. **Замініть плейсхолдери** у текстах: `[НАЗВА_ОПЕРАТОРА]`, `[EMAIL_ПІДТРИМКИ]` тощо (або внесіть реквізити, яких вимагає законодавство країни реєстрації оператора).
2. **Звірка з юристом**: юрисдикція, застосування GDPR якщо є користувачі з ЄС/ЄЕЗ, право обране в договорі, вік дітей, претензії / ВОС, особливості **окупованих територій** тощо — **не автоматизуються** цим репозиторієм.
3. **App Store / Google Play**: у консолях часто потрібні **URL** на опубліковану політику конфіденційності й умови; після деплою сайту вкажіть стабільні посилання (наприклад `https://…/privacy`).
4. **Оновлення**: змінюєте збір даних (нові поля, реклама, аналітика, треті сторони) — оновіть **PRIVACY** і дату внизу документа.

## Рекомендації щодо валідації юристом (короткий чеклист)

- Хто є **власником персональних даних** (контролер) юридично: ФОП, ТОВ, нерезидент.
- Чи потрібна **реєстрація обробки** в українського Уповноваженого (закон України «Про захист персональних даних») та **типові заходи**.
- Якщо є **користувачі з ЄС** — **GDPR**: право на видалення/експорт, правові підстави обробки, DPA з підпроцесором (хостинг, Telegram API).
- **Платні функції / freemium / платний додаток** — оновити **TERMS** (оплата, відмова, повернення згідно з правилами Apple/Google та національним законом).
- Чи сервіс адресований **дітям** (вік 13/16) — тоді окремі вимоги.

Після правок юриста зафіксуйте зміну в **[../CHANGELOG.md](../CHANGELOG.md)** і за потреби у **[../ROADMAP.md](../ROADMAP.md)**.
